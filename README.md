# Photography YouTube Channel Website

A modern, single-page website for a photography YouTube channel.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Video showcase section
- Social media integration
- Modern gradient design
- Scroll animations

## GitHub Pages Setup

To publish this site to GitHub Pages:

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" in the left sidebar
4. Under "Source", select your branch (usually `main` or `master`)
5. Select the root folder (`/`)
6. Click "Save"
7. Your site will be available at `https://yourusername.github.io/repository-name/`

## Customization

### Update YouTube Links
Replace all instances of `https://youtube.com` in `index.html` with your actual YouTube channel URL.

### Update Social Media Links
Update the social media links in the contact section with your actual profiles:
- YouTube channel URL
- Instagram profile URL
- Twitter/X profile URL
- Email address

### Update Channel Information
- Modify the channel name, description, and stats in the About section
- Update video titles and descriptions in the Videos section
- Replace placeholder content with your actual content

### Add Real Video Thumbnails
Replace the gradient placeholders in `.video-thumbnail` divs with actual YouTube video thumbnail images.
